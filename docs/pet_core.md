# 宠物核心规则

## 宠物档案

每只宠物都拥有独立档案，至少包括：

- `pet_id`
- `name`
- `species_id`
- `theme_id`
- `growth_stage`
- `level`
- `exp`
- `mood`
- `energy`
- `hunger`
- `affection`
- `owner_id`
- `device_id`

## 记忆分层

### 短期记忆

用于最近互动上下文，例如最近几句对话、刚刚发生的动作。

### 长期记忆

用于稳定偏好，例如喜欢的称呼、常见作息、固定喜好。

### 事件记忆

用于一次性的重要事件，例如首次绑定、升级、节日、异常恢复。

## 成长规则

- 互动可增长经验
- 经验达到阈值后升级
- 升级可改变成长阶段
- 成长阶段决定 UI 和动作表现

## 情绪规则

- 能量低：偏 sleepy
- 饥饿高：偏 hungry
- 绑定成功/互动积极：偏 happy
- 未绑定或长时间无互动：偏 lonely / curious

## 后续扩展

- 事件可按类别汇总
- 宠物事件可输出最近摘要
- 节日事件奖励
- 行为习惯形成
