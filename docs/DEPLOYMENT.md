# 部署指南

## 环境要求
- Python 3.7+
- 2GB RAM
- 网络连接

## 快速开始

### Windows
```bash
setup.bat
start.bat
```

### Linux/Mac
```bash
chmod +x setup.sh start.sh
./setup.sh
./start.sh
```

## 访问应用
- API 文档: http://localhost:8000/docs
- Web 界面: http://localhost:8000/

## 配置 LLM
访问 API 文档配置您的 LLM 提供商（OpenAI、Claude 等）。
