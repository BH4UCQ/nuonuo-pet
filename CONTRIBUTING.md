# 贡献指南

感谢您对 nuonuo-pet 项目的关注！

## 如何贡献

### 报告问题
如果您发现了 bug 或有功能建议，请在 Issues 中详细描述。

### 提交代码
1. Fork 项目
2. 创建特性分支
3. 提交更改
4. 推送到您的 Fork
5. 创建 Pull Request

### 代码规范
- 遵循 PEP 8 代码风格
- 添加必要的测试
- 更新相关文档

## 开发环境设置

1. 克隆项目
2. 创建虚拟环境
3. 安装依赖
4. 运行开发服务器

详细步骤请参考 README.md。

## 许可证

通过贡献代码，您同意您的贡献将在 MIT 许可证下发布。
